<p>JavaScript 中的**可迭代对象（Iterables）<strong>和</strong>生成器（Generators）**是处理序列数据的核心机制，它们让异步操作、惰性计算和数据流处理变得优雅。</p>

<h2>一、可迭代对象</h2>
<p>如果一个对象实现了 <strong><code>[Symbol.iterator]</code></strong> 方法，并且该方法返回一个遵循<strong>迭代器协议</strong>的对象（即具有 <code>next()</code> 方法，返回 <code>{ value, done }</code>），那么它就是可迭代的。</p>
<h3>内置的可迭代对象</h3>
<p>JavaScript 内置了多种可迭代对象，可以直接使用 <code>for...of</code> 遍历：</p>
<pre class="shiki github-dark" style="background-color:#24292e;color:#e1e4e8" tabindex="0"><code><span class="line"><span style="color:#6A737D">// 字符串 —— 按 Unicode 字符遍历</span></span>
<span class="line"><span style="color:#F97583">for</span><span style="color:#E1E4E8"> (</span><span style="color:#F97583">const</span><span style="color:#79B8FF"> char</span><span style="color:#F97583"> of</span><span style="color:#9ECBFF"> "hello"</span><span style="color:#E1E4E8">) {</span></span>
<span class="line"><span style="color:#E1E4E8">  console.</span><span style="color:#B392F0">log</span><span style="color:#E1E4E8">(char); </span><span style="color:#6A737D">// h, e, l, l, o</span></span>
<span class="line"><span style="color:#E1E4E8">}</span></span>
<span class="line"></span>
<span class="line"><span style="color:#6A737D">// 数组</span></span>

<span class="line"><span style="color:#F97583">for</span><span style="color:#E1E4E8"> (</span><span style="color:#F97583">const</span><span style="color:#E1E4E8"> [</span><span style="color:#79B8FF">key</span><span style="color:#E1E4E8">, </span><span style="color:#79B8FF">value</span><span style="color:#E1E4E8">] </span><span style="color:#F97583">of</span><span style="color:#E1E4E8"> map) {</span></span>

<h3><code>for...of</code> 的底层原理</h3>
<p><code>for...of</code> 本质上是这样工作的：</p>
<pre class="shiki github-dark" style="background-color:#24292e;color:#e1e4e8" tabindex="0"><code><span class="line"><span style="color:#F97583">const</span><span style="color:#79B8FF"> iterable</span><span style="color:#F97583"> =</span><span style="color:#E1E4E8"> [</span><span style="color:#79B8FF">10</span><span style="color:#E1E4E8">, </span><span style="color:#79B8FF">20</span><span style="color:#E1E4E8">, </span><span style="color:#79B8FF">30</span><span style="color:#E1E4E8">];</span></span>
<span class="line"></span>
<span class="line"><span style="color:#6A737D">// 1. 获取迭代器</span></span>
<span class="line"><span style="color:#F97583">const</span><span style="color:#79B8FF"> iterator</span><span style="color:#F97583"> =</span><span style="color:#E1E4E8"> iterable[Symbol.iterator]();</span></span>
<span class="line"></span>
<span class="line"><span style="color:#6A737D">// 2. 反复调用 next()，直到 done 为 true</span></span>
<span class="line"><span style="color:#F97583">let</span><span style="color:#E1E4E8"> result </span><span style="color:#F97583">=</span><span style="color:#E1E4E8"> iterator.</span><span style="color:#B392F0">next</span><span style="color:#E1E4E8">();</span></span>
<span class="line"><span style="color:#F97583">while</span><span style="color:#E1E4E8"> (</span><span style="color:#F97583">!</span><span style="color:#E1E4E8">result.done) {</span></span>
<span class="line"><span style="color:#E1E4E8">  console.</span><span style="color:#B392F0">log</span><span style="color:#E1E4E8">(result.value); </span><span style="color:#6A737D">// 10, 20, 30</span></span>
<span class="line"><span style="color:#E1E4E8">  result </span><span style="color:#F97583">=</span><span style="color:#E1E4E8"> iterator.</span><span style="color:#B392F0">next</span><span style="color:#E1E4E8">();</span></span>
<span class="line"><span style="color:#E1E4E8">}</span></span></code></pre>
<p>理解这个机制，有助于你更好地自定义可迭代行为。</p>

<h2>二、生成器</h2>
<p>生成器是 ES6 引入的特殊函数，用 <code>function*</code> 声明。它最大的特点是<strong>可以在执行过程中暂停和恢复</strong>，通过 <code>yield</code> 关键字交出控制权。</p>
<h3>基本语法</h3>
<pre class="shiki github-dark" style="background-color:#24292e;color:#e1e4e8" tabindex="0"><code><span class="line"><span style="color:#F97583">function*</span><span style="color:#B392F0"> idGenerator</span><span style="color:#E1E4E8">() {</span></span>
<span class="line"><span style="color:#F97583">  let</span><span style="color:#E1E4E8"> id </span><span style="color:#F97583">=</span><span style="color:#79B8FF"> 1</span><span style="color:#E1E4E8">;</span></span>
<span class="line"><span style="color:#F97583">  while</span><span style="color:#E1E4E8"> (</span><span style="color:#79B8FF">true</span><span style="color:#E1E4E8">) {</span></span>
<span class="line"><span style="color:#F97583">    yield</span><span style="color:#E1E4E8"> id</span><span style="color:#F97583">++</span><span style="color:#E1E4E8">; </span><span style="color:#6A737D">// 每次在这里暂停，返回当前 id</span></span>
<span class="line"><span style="color:#E1E4E8">  }</span></span>
<span class="line"><span style="color:#E1E4E8">}</span></span>
<span class="line"></span>
<span class="line"><span style="color:#F97583">const</span><span style="color:#79B8FF"> gen</span><span style="color:#F97583"> =</span><span style="color:#B392F0"> idGenerator</span><span style="color:#E1E4E8">();</span></span>
<span class="line"><span style="color:#E1E4E8">console.</span><span style="color:#B392F0">log</span><span style="color:#E1E4E8">(gen.</span><span style="color:#B392F0">next</span><span style="color:#E1E4E8">().value); </span><span style="color:#6A737D">// 1</span></span>
<span class="line"><span style="color:#E1E4E8">console.</span><span style="color:#B392F0">log</span><span style="color:#E1E4E8">(gen.</span><span style="color:#B392F0">next</span><span style="color:#E1E4E8">().value); </span><span style="color:#6A737D">// 2</span></span>
<span class="line"><span style="color:#E1E4E8">console.</span><span style="color:#B392F0">log</span><span style="color:#E1E4E8">(gen.</span><span style="color:#B392F0">next</span><span style="color:#E1E4E8">().value); </span><span style="color:#6A737D">// 3</span></span>
<span class="line"><span style="color:#6A737D">// 不会无限循环，每次 yield 都会把控制权交还调用方</span></span></code></pre>
<h3>生成器与可迭代的关系</h3>
<p><strong>生成器对象同时满足迭代器协议和可迭代协议</strong>——它既是迭代器（有 <code>next()</code> 方法），也是可迭代对象（<code>[Symbol.iterator]</code> 返回自身）。这意味着生成器可以无缝接入任何需要可迭代对象的场景。</p>
<pre class="shiki github-dark" style="background-color:#24292e;color:#e1e4e8" tabindex="0"><code><span class="line"><span style="color:#F97583">function*</span><span style="color:#B392F0"> countdown</span><span style="color:#E1E4E8">(</span><span style="color:#FFAB70">n</span><span style="color:#E1E4E8">) {</span></span>
<span class="line"><span style="color:#F97583">  while</span><span style="color:#E1E4E8"> (n </span><span style="color:#F97583">>=</span><span style="color:#79B8FF"> 0</span><span style="color:#E1E4E8">) {</span></span>
<span class="line"><span style="color:#F97583">    yield</span><span style="color:#E1E4E8"> n;</span></span>
<span class="line"><span style="color:#E1E4E8">    n</span><span style="color:#F97583">--</span><span style="color:#E1E4E8">;</span></span>
<span class="line"><span style="color:#E1E4E8">  }</span></span>
<span class="line"><span style="color:#E1E4E8">}</span></span>
<span class="line"></span>
<span class="line"><span style="color:#6A737D">// 生成器可以直接用于 for...of</span></span>
<span class="line"><span style="color:#F97583">for</span><span style="color:#E1E4E8"> (</span><span style="color:#F97583">const</span><span style="color:#79B8FF"> num</span><span style="color:#F97583"> of</span><span style="color:#B392F0"> countdown</span><span style="color:#E1E4E8">(</span><span style="color:#79B8FF">5</span><span style="color:#E1E4E8">)) {</span></span>
<span class="line"><span style="color:#E1E4E8">  console.</span><span style="color:#B392F0">log</span><span style="color:#E1E4E8">(num); </span><span style="color:#6A737D">// 5, 4, 3, 2, 1, 0</span></span>
<span class="line"><span style="color:#E1E4E8">}</span></span>
<span class="line"></span>
<span class="line"><span style="color:#6A737D">// 也可以展开</span></span>
<span class="line"><span style="color:#E1E4E8">console.</span><span style="color:#B392F0">log</span><span style="color:#E1E4E8">([</span><span style="color:#F97583">...</span><span style="color:#B392F0">countdown</span><span style="color:#E1E4E8">(</span><span style="color:#79B8FF">3</span><span style="color:#E1E4E8">)]); </span><span style="color:#6A737D">// [3, 2, 1, 0]</span></span>
<span class="line"></span>
<span class="line"><span style="color:#6A737D">// 还能解构</span></span>
<span class="line"><span style="color:#F97583">const</span><span style="color:#E1E4E8"> [</span><span style="color:#79B8FF">first</span><span style="color:#E1E4E8">, </span><span style="color:#79B8FF">second</span><span style="color:#E1E4E8">] </span><span style="color:#F97583">=</span><span style="color:#B392F0"> countdown</span><span style="color:#E1E4E8">(</span><span style="color:#79B8FF">10</span><span style="color:#E1E4E8">);</span></span>
<span class="line"><span style="color:#E1E4E8">console.</span><span style="color:#B392F0">log</span><span style="color:#E1E4E8">(first, second); </span><span style="color:#6A737D">// 10, 9</span></span></code></pre>
<h3><code>return</code> 在生成器中的特殊行为</h3>
<p>生成器内的 <code>return</code> 会结束迭代，将 <code>done</code> 设为 <code>true</code>：</p>
<pre class="shiki github-dark" style="background-color:#24292e;color:#e1e4e8" tabindex="0"><code><span class="line"><span style="color:#F97583">function*</span><span style="color:#B392F0"> withReturn</span><span style="color:#E1E4E8">() {</span></span>
<span class="line"><span style="color:#F97583">  yield</span><span style="color:#79B8FF"> 1</span><span style="color:#E1E4E8">;</span></span>
<span class="line"><span style="color:#F97583">  yield</span><span style="color:#79B8FF"> 2</span><span style="color:#E1E4E8">;</span></span>
<span class="line"><span style="color:#F97583">  return</span><span style="color:#79B8FF"> 3</span><span style="color:#E1E4E8">;  </span><span style="color:#6A737D">// 结束生成器，value 3 通常被忽略</span></span>
<span class="line"><span style="color:#F97583">  yield</span><span style="color:#79B8FF"> 4</span><span style="color:#E1E4E8">;   </span><span style="color:#6A737D">// 不会执行</span></span>
<span class="line"><span style="color:#E1E4E8">}</span></span>
<span class="line"></span>
<span class="line"><span style="color:#F97583">const</span><span style="color:#79B8FF"> g</span><span style="color:#F97583"> =</span><span style="color:#B392F0"> withReturn</span><span style="color:#E1E4E8">();</span></span>
<span class="line"><span style="color:#E1E4E8">console.</span><span style="color:#B392F0">log</span><span style="color:#E1E4E8">(g.</span><span style="color:#B392F0">next</span><span style="color:#E1E4E8">()); </span><span style="color:#6A737D">// { value: 1, done: false }</span></span>
<span class="line"><span style="color:#E1E4E8">console.</span><span style="color:#B392F0">log</span><span style="color:#E1E4E8">(g.</span><span style="color:#B392F0">next</span><span style="color:#E1E4E8">()); </span><span style="color:#6A737D">// { value: 2, done: false }</span></span>
<span class="line"><span style="color:#E1E4E8">console.</span><span style="color:#B392F0">log</span><span style="color:#E1E4E8">(g.</span><span style="color:#B392F0">next</span><span style="color:#E1E4E8">()); </span><span style="color:#6A737D">// { value: 3, done: true } ← return 的值</span></span>
<span class="line"><span style="color:#E1E4E8">console.</span><span style="color:#B392F0">log</span><span style="color:#E1E4E8">(g.</span><span style="color:#B392F0">next</span><span style="color:#E1E4E8">()); </span><span style="color:#6A737D">// { value: undefined, done: true }</span></span></code></pre>
<blockquote>
<p>注意：<code>return</code> 的值虽然会出现在最后一个 <code>next()</code> 的结果中，但 <code>for...of</code> 和展开运算符会忽略它。</p>
</blockquote>
<h3>生成器的 <code>return()</code> 和 <code>throw()</code></h3>
<p>作为调用方，你可以主动控制生成器的生命周期：</p>
<pre class="shiki github-dark" style="background-color:#24292e;color:#e1e4e8" tabindex="0"><code><span class="line"><span style="color:#F97583">function*</span><span style="color:#B392F0"> gen</span><span style="color:#E1E4E8">() {</span></span>
<span class="line"><span style="color:#F97583">  try</span><span style="color:#E1E4E8"> {</span></span>
<span class="line"><span style="color:#F97583">    yield</span><span style="color:#79B8FF"> 1</span><span style="color:#E1E4E8">;</span></span>
<span class="line"><span style="color:#F97583">    yield</span><span style="color:#79B8FF"> 2</span><span style="color:#E1E4E8">;</span></span>
<span class="line"><span style="color:#F97583">    yield</span><span style="color:#79B8FF"> 3</span><span style="color:#E1E4E8">;</span></span>
<span class="line"><span style="color:#E1E4E8">  } </span><span style="color:#F97583">finally</span><span style="color:#E1E4E8"> {</span></span>
<span class="line"><span style="color:#E1E4E8">    console.</span><span style="color:#B392F0">log</span><span style="color:#E1E4E8">(</span><span style="color:#9ECBFF">'清理资源'</span><span style="color:#E1E4E8">);  </span><span style="color:#6A737D">// finally 会在生成器被终止时执行</span></span>
<span class="line"><span style="color:#E1E4E8">  }</span></span>
<span class="line"><span style="color:#E1E4E8">}</span></span>
<span class="line"></span>
<span class="line"><span style="color:#F97583">const</span><span style="color:#79B8FF"> g</span><span style="color:#F97583"> =</span><span style="color:#B392F0"> gen</span><span style="color:#E1E4E8">();</span></span>
<span class="line"><span style="color:#E1E4E8">console.</span><span style="color:#B392F0">log</span><span style="color:#E1E4E8">(g.</span><span style="color:#B392F0">next</span><span style="color:#E1E4E8">());        </span><span style="color:#6A737D">// { value: 1, done: false }</span></span>
<span class="line"><span style="color:#E1E4E8">console.</span><span style="color:#B392F0">log</span><span style="color:#E1E4E8">(g.</span><span style="color:#B392F0">return</span><span style="color:#E1E4E8">(</span><span style="color:#79B8FF">100</span><span style="color:#E1E4E8">));   </span><span style="color:#6A737D">// { value: 100, done: true } —— 提前终止</span></span>
<span class="line"><span style="color:#6A737D">// 输出："清理资源"</span></span>
<span class="line"></span>
<span class="line"><span style="color:#F97583">const</span><span style="color:#79B8FF"> g2</span><span style="color:#F97583"> =</span><span style="color:#B392F0"> gen</span><span style="color:#E1E4E8">();</span></span>
<span class="line"><span style="color:#E1E4E8">g2.</span><span style="color:#B392F0">next</span><span style="color:#E1E4E8">();                    </span><span style="color:#6A737D">// 先启动到第一个 yield</span></span>
<span class="line"><span style="color:#E1E4E8">g2.</span><span style="color:#B392F0">throw</span><span style="color:#E1E4E8">(</span><span style="color:#F97583">new</span><span style="color:#B392F0"> Error</span><span style="color:#E1E4E8">(</span><span style="color:#9ECBFF">'出错了'</span><span style="color:#E1E4E8">)); </span><span style="color:#6A737D">// 向生成器内部注入异常</span></span>
<span class="line"><span style="color:#6A737D">// Error: 出错了</span></span></code></pre>
<p>这两个方法在需要<strong>提前终止生成器</strong>或<strong>处理生成器内部异常</strong>时非常有用。</p>
<h3><code>yield*</code> —— 委托迭代</h3>
<p><code>yield*</code> 可以将迭代工作委托给另一个可迭代对象，避免手动逐个 <code>yield</code>：</p>
<pre class="shiki github-dark" style="background-color:#24292e;color:#e1e4e8" tabindex="0"><code><span class="line"><span style="color:#F97583">function*</span><span style="color:#B392F0"> combined</span><span style="color:#E1E4E8">() {</span></span>
<span class="line"><span style="color:#F97583">  yield*</span><span style="color:#E1E4E8"> [</span><span style="color:#79B8FF">1</span><span style="color:#E1E4E8">, </span><span style="color:#79B8FF">2</span><span style="color:#E1E4E8">, </span><span style="color:#79B8FF">3</span><span style="color:#E1E4E8">];</span></span>
<span class="line"><span style="color:#F97583">  yield</span><span style="color:#9ECBFF"> 'end'</span><span style="color:#E1E4E8">;</span></span>
<span class="line"><span style="color:#E1E4E8">}</span></span>
<span class="line"></span>
<span class="line"><span style="color:#6A737D">// 上面的代码等效于：</span></span>
<span class="line"><span style="color:#F97583">function*</span><span style="color:#B392F0"> combinedManual</span><span style="color:#E1E4E8">() {</span></span>
<span class="line"><span style="color:#F97583">  yield</span><span style="color:#79B8FF"> 1</span><span style="color:#E1E4E8">;</span></span>
<span class="line"><span style="color:#F97583">  yield</span><span style="color:#79B8FF"> 2</span><span style="color:#E1E4E8">;</span></span>
<span class="line"><span style="color:#F97583">  yield</span><span style="color:#79B8FF"> 3</span><span style="color:#E1E4E8">;</span></span>
<span class="line"><span style="color:#F97583">  yield</span><span style="color:#9ECBFF"> 'end'</span><span style="color:#E1E4E8">;</span></span>
<span class="line"><span style="color:#E1E4E8">}</span></span>
<span class="line"></span>
<span class="line"><span style="color:#E1E4E8">console.</span><span style="color:#B392F0">log</span><span style="color:#E1E4E8">([</span><span style="color:#F97583">...</span><span style="color:#B392F0">combined</span><span style="color:#E1E4E8">()]); </span><span style="color:#6A737D">// [1, 2, 3, 'end']</span></span></code></pre>
<p><code>yield*</code> 还可以递归委托给另一个生成器，常用于遍历树形结构。</p>
<h3>双向通信</h3>
<p>生成器不仅可以向外产出值，还可以通过 <code>next(value)</code> 向内部传入数据：</p>
<pre class="shiki github-dark" style="background-color:#24292e;color:#e1e4e8" tabindex="0"><code><span class="line"><span style="color:#F97583">function*</span><span style="color:#B392F0"> chat</span><span style="color:#E1E4E8">() {</span></span>
<span class="line"><span style="color:#F97583">  const</span><span style="color:#79B8FF"> name</span><span style="color:#F97583"> =</span><span style="color:#F97583"> yield</span><span style="color:#9ECBFF"> "你叫什么名字？"</span><span style="color:#E1E4E8">;</span></span>
<span class="line"><span style="color:#F97583">  yield</span><span style="color:#9ECBFF"> `你好，${</span><span style="color:#E1E4E8">name</span><span style="color:#9ECBFF">}！`</span><span style="color:#E1E4E8">;</span></span>
<span class="line"><span style="color:#E1E4E8">}</span></span>
<span class="line"></span>
<span class="line"><span style="color:#F97583">const</span><span style="color:#79B8FF"> g</span><span style="color:#F97583"> =</span><span style="color:#B392F0"> chat</span><span style="color:#E1E4E8">();</span></span>
<span class="line"><span style="color:#E1E4E8">console.</span><span style="color:#B392F0">log</span><span style="color:#E1E4E8">(g.</span><span style="color:#B392F0">next</span><span style="color:#E1E4E8">().value);         </span><span style="color:#6A737D">// "你叫什么名字？"</span></span>
<span class="line"><span style="color:#E1E4E8">console.</span><span style="color:#B392F0">log</span><span style="color:#E1E4E8">(g.</span><span style="color:#B392F0">next</span><span style="color:#E1E4E8">(</span><span style="color:#9ECBFF">"Alice"</span><span style="color:#E1E4E8">).value);  </span><span style="color:#6A737D">// "你好，Alice！"</span></span></code></pre>
<p>第一次调用 <code>g.next()</code> 启动生成器，之后每次传入的参数会成为上一个 <code>yield</code> 表达式的返回值。这种双向通信让生成器可以实现<strong>协程</strong>模式。</p>
<hr>
<h2>三、异步生成器</h2>
<p>ES2018 引入了异步生成器（<code>async function*</code>）和 <code>for await...of</code> 语法，用于处理异步数据流。它的核心是：</p>
<ul>
<li>用 <code>async function*</code> 定义异步生成器</li>
<li>内部可以 <code>await</code> 异步操作，也可以 <code>yield</code> 值</li>
<li>外部用 <code>for await...of</code> 消费</li>
</ul>
<pre class="shiki github-dark" style="background-color:#24292e;color:#e1e4e8" tabindex="0"><code><span class="line"><span style="color:#6A737D">// 异步生成器：每 500ms 产出一个值</span></span>
<span class="line"><span style="color:#F97583">async</span><span style="color:#F97583"> function*</span><span style="color:#B392F0"> asyncCounter</span><span style="color:#E1E4E8">() {</span></span>
<span class="line"><span style="color:#F97583">  let</span><span style="color:#E1E4E8"> i </span><span style="color:#F97583">=</span><span style="color:#79B8FF"> 0</span><span style="color:#E1E4E8">;</span></span>
<span class="line"><span style="color:#F97583">  while</span><span style="color:#E1E4E8"> (i </span><span style="color:#F97583">&#x3C;</span><span style="color:#79B8FF"> 3</span><span style="color:#E1E4E8">) {</span></span>
<span class="line"><span style="color:#F97583">    await</span><span style="color:#F97583"> new</span><span style="color:#79B8FF"> Promise</span><span style="color:#E1E4E8">(</span><span style="color:#FFAB70">resolve</span><span style="color:#F97583"> =></span><span style="color:#B392F0"> setTimeout</span><span style="color:#E1E4E8">(resolve, </span><span style="color:#79B8FF">500</span><span style="color:#E1E4E8">));</span></span>
<span class="line"><span style="color:#F97583">    yield</span><span style="color:#E1E4E8"> i</span><span style="color:#F97583">++</span><span style="color:#E1E4E8">;</span></span>
<span class="line"><span style="color:#E1E4E8">  }</span></span>
<span class="line"><span style="color:#E1E4E8">}</span></span>
<span class="line"></span>
<span class="line"><span style="color:#6A737D">// 消费异步生成器</span></span>
<span class="line"><span style="color:#E1E4E8">(</span><span style="color:#F97583">async</span><span style="color:#E1E4E8"> () </span><span style="color:#F97583">=></span><span style="color:#E1E4E8"> {</span></span>
<span class="line"><span style="color:#F97583">  for</span><span style="color:#F97583"> await</span><span style="color:#E1E4E8"> (</span><span style="color:#F97583">const</span><span style="color:#79B8FF"> num</span><span style="color:#F97583"> of</span><span style="color:#B392F0"> asyncCounter</span><span style="color:#E1E4E8">()) {</span></span>
<span class="line"><span style="color:#E1E4E8">    console.</span><span style="color:#B392F0">log</span><span style="color:#E1E4E8">(num); </span><span style="color:#6A737D">// 0, 1, 2（每 500ms 输出一个）</span></span>
<span class="line"><span style="color:#E1E4E8">  }</span></span>
<span class="line"><span style="color:#E1E4E8">})();</span></span></code></pre>
<p>异步生成器是连接"异步操作"与"迭代器模式"的桥梁，也是接下来两个实战场景的基础。</p>

<h2>四、实际应用场景</h2>
<p>掌握基础概念后，我们来看看生成器在真实项目中如何解决实际问题。</p>
<h3>1. 无限滚动</h3>

<h3>2. 流式传输（SSE）</h3>
<p>处理 ChatGPT 这类流式接口时，异步生成器可以将底层的 SSE（Server-Sent Events）格式封装成干净的数据流：</p>
<pre class="shiki github-dark" style="background-color:#24292e;color:#e1e4e8" tabindex="0"><code><span class="line"><span style="color:#6A737D">// 封装层：把 fetch SSE 流转成异步生成器</span></span>

<span class="line"><span style="color:#E1E4E8">    body: </span><span style="color:#79B8FF">JSON</span><span style="color:#E1E4E8">.</span><span style="color:#B392F0">stringify</span><span style="color:#E1E4E8">({ messages, stream: </span><span style="color:#79B8FF">true</span><span style="color:#E1E4E8"> })</span></span>
<span class="line"><span style="color:#E1E4E8">  });</span></span>
<span class="line"></span>
<span class="line"><span style="color:#F97583">  const</span><span style="color:#79B8FF"> reader</span><span style="color:#F97583"> =</span><span style="color:#E1E4E8"> response.body.</span><span style="color:#B392F0">getReader</span><span style="color:#E1E4E8">();</span></span>
<span class="line"><span style="color:#F97583">  const</span><span style="color:#79B8FF"> decoder</span><span style="color:#F97583"> =</span><span style="color:#F97583"> new</span><span style="color:#B392F0"> TextDecoder</span><span style="color:#E1E4E8">();</span></span>

<span class="line"><span style="color:#F97583">  try</span><span style="color:#E1E4E8"> {</span></span>

<span class="line"><span style="color:#F97583">      const</span><span style="color:#E1E4E8"> { </span><span style="color:#79B8FF">done</span><span style="color:#E1E4E8">, </span><span style="color:#79B8FF">value</span><span style="color:#E1E4E8"> } </span><span style="color:#F97583">=</span><span style="color:#F97583"> await</span><span style="color:#E1E4E8"> reader.</span><span style="color:#B392F0">read</span><span style="color:#E1E4E8">();</span></span>
<span class="line"><span style="color:#F97583">      if</span><span style="color:#E1E4E8"> (done) </span><span style="color:#F97583">break</span><span style="color:#E1E4E8">;</span></span>
<span class="line"></span>
<span class="line"><span style="color:#F97583">      const</span><span style="color:#79B8FF"> chunk</span><span style="color:#F97583"> =</span><span style="color:#E1E4E8"> decoder.</span><span style="color:#B392F0">decode</span><span style="color:#E1E4E8">(value, { stream: </span><span style="color:#79B8FF">true</span><span style="color:#E1E4E8"> });</span></span>
<span class="line"><span style="color:#6A737D">      // SSE 格式：每行以 "data: " 开头</span></span>
<span class="line"><span style="color:#F97583">      for</span><span style="color:#E1E4E8"> (</span><span style="color:#F97583">const</span><span style="color:#79B8FF"> line</span><span style="color:#F97583"> of</span><span style="color:#E1E4E8"> chunk.</span><span style="color:#B392F0">split</span><span style="color:#E1E4E8">(</span><span style="color:#9ECBFF">'</span><span style="color:#79B8FF">\n</span><span style="color:#9ECBFF">'</span><span style="color:#E1E4E8">)) {</span></span>
